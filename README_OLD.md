# نحكي - منصة التواصل الاجتماعي العربية

## نظرة عامة

نحكي هي منصة تواصل اجتماعي عربية مبتكرة مصممة خصيصاً لأجهزة الكمبيوتر. تهدف المنصة إلى تعزيز العلاقة بين المؤثرين والمتابعين من خلال نظام "كبار المتابعين" ونظام النقاط التفاعلي.

## الميزات الرئيسية

### 🎯 نظام كبار المتابعين
- يمكن للمؤثرين (أكثر من 1000 متابع) ترشيح 3 متابعين كحد أقصى كل أسبوعين
- كبار المتابعين يمكنهم النشر/التعديل في صفحة المؤثر بموافقته
- صلاحية محدودة: أسبوعين أو 3 منشورات (أيهما أولاً)

### ⭐ نظام النقاط
- **20 نقطة**: نشر منشور باسم المؤثر
- **10 نقاط**: كتابة تعليق
- **5 نقاط**: إعجاب
- النقاط تُحسب لكل مؤثر منفصلاً

### 🔐 نظام التسجيل
- تسجيل الدخول برقم الهاتف أو البريد الإلكتروني أو اسم المستخدم
- التحقق عبر رمز OTP

### 💬 التفاعل والمحتوى
- منشورات نصية وصور وفيديوهات
- تعليقات وإعجابات وإعادة نشر
- رسائل خاصة وتبادل الملفات
- إشعارات لحظية

## التقنيات المستخدمة

### Frontend
- **Next.js 14** - إطار عمل React
- **TypeScript** - لتطوير أكثر أماناً
- **Tailwind CSS** - للتصميم والتنسيق
- **دعم RTL كامل** للغة العربية

### Backend
- **Node.js + Express.js** - خادم التطبيق
- **PostgreSQL** - قاعدة البيانات الرئيسية
- **Prisma ORM** - إدارة قاعدة البيانات
- **Socket.IO** - التفاعل اللحظي

### المصادقة والأمان
- **JWT** - إدارة جلسات المستخدمين
- **OTP** - التحقق عبر SMS/Email
- **bcrypt** - تشفير كلمات المرور


## التثبيت والتشغيل

### المتطلبات
- Node.js 18+ 
- PostgreSQL 13+
- npm أو yarn

### خطوات التثبيت

1. **استنساخ المشروع**
   ```bash
   git clone [repository-url]
   cd nehky-social-platform
   ```

2. **تثبيت المكتبات**
   ```bash
   npm install
   ```

3. **إعداد متغيرات البيئة**
   ```bash
   cp .env.example .env.local
   ```
   قم بتحديث `.env.local` بالقيم الصحيحة

4. **إعداد قاعدة البيانات**
   ```bash
   npx prisma generate
   npx prisma db push
   ```

5. **تشغيل المشروع**
   ```bash
   npm run dev
   ```

6. **افتح المتصفح**
   ```
   http://localhost:3000
   ```

## قاعدة البيانات

### الجداول الرئيسية
- **Users**: معلومات المستخدمين والملفات الشخصية
- **Posts**: المنشورات والمحتوى
- **Comments**: التعليقات
- **Likes**: الإعجابات
- **Points**: نظام النقاط
- **Nominations**: ترشيحات كبار المتابعين
- **Messages**: الرسائل الخاصة
- **Follows**: علاقات المتابعة

### الأدوار (Roles)
- `USER`: مستخدم عادي
- `INFLUENCER`: مؤثر (أكثر من 1000 متابع)
- `TOP_FOLLOWER`: كبير متابعين (مرشح من مؤثر)

## API Routes

### المصادقة
- `POST /api/auth/register` - إنشاء حساب جديد
- `POST /api/auth/send-otp` - إرسال رمز التحقق
- `POST /api/auth/verify-otp` - التحقق من رمز OTP

### المستخدمين
- `GET /api/users/profile` - الحصول على الملف الشخصي
- `PUT /api/users/profile` - تحديث الملف الشخصي
- `POST /api/users/follow` - متابعة مستخدم

### المنشورات
- `GET /api/posts` - الحصول على المنشورات
- `POST /api/posts` - إنشاء منشور جديد
- `POST /api/posts/like` - إعجاب بمنشور
- `POST /api/posts/comment` - تعليق على منشور

### النقاط والترشيحات
- `GET /api/points` - الحصول على نقاط المستخدم
- `POST /api/nominations` - ترشيح متابع
- `GET /api/nominations` - الحصول على الترشيحات

## التصميم والألوان

### الشعار
- **الشعار الرسمي**: `/public/nehky_logo.webp`
- **مكون الشعار**: `@/components/Logo`
- **الاستخدام**: أحجام متعددة (sm, md, lg, xl) مع إمكانية إظهار/إخفاء النص

### نظام الألوان
- **Primary**: تدرج من الأزرق (#1e40af) إلى الأخضر (#059669)
- **Secondary**: درجات الأزرق والأخضر المختلفة
- **Accent**: ألوان تكميلية للتأكيد

### الخطوط
- **Arabic**: Cairo, Tajawal
- **Fallback**: system-ui, sans-serif

### دعم RTL
- دعم كامل للغة العربية من اليمين إلى اليسار
- تخطيط مرن يتكيف مع اتجاه النص
