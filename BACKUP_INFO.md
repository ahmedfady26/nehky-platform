## التقنيات المستخدمة
- **Frontend**: Next.js 14 مع TypeScript و Tailwind CSS
- **Backend**: Node.js + Express.js
- **Database**: PostgreSQL مع Prisma ORM
- **Real-time**: Socket.IO للإشعارات اللحظية
- **Authentication**: OTP عبر SMS/Email

## الملفات الهامة
- `prisma/schema.prisma`: نموذج قاعدة البيانات المحدث
- `src/lib/points-system.ts`: منطق نظام النقاط والترشيحات
- `src/app/points/page.tsx`: واجهة عرض النقاط والترشيحات
- `src/components/PostInteraction.tsx`: مكون التفاعل مع المنشورات
- جميع API routes في `src/app/api/`

- النظام جاهز للإنتاج بعد ضبط متغيرات البيئة

## كيفية استرداد النسخة
1. انسخ المجلد إلى موقع المشروع الجديد
2. شغل `npm install` لتحديث node_modules
3. انسخ ملف `.env.local` وضبط متغيرات البيئة
4. شغل `npx prisma db push` لإعداد قاعدة البيانات
5. شغل `npm run dev` لبدء المشروع
