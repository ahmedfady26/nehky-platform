# نظام الإشعارات - تعليمات التشغيل

## 🚀 تشغيل خادم الإشعارات

### 1. انتقل إلى مجلد الخادم
```bash
cd server
```

### 2. تثبيت المكتبات المطلوبة
```bash
npm install
```

### 3. تشغيل خادم الإشعارات
```bash
# للتطوير مع إعادة التشغيل التلقائي
npm run dev

# أو للتشغيل العادي
npm start
```

الخادم سيعمل على المنفذ 3001: `http://localhost:3001`

## 🔧 تشغيل المنصة الرئيسية

### 1. في نافذة terminal منفصلة، ارجع للمجلد الرئيسي
```bash
cd ..
```

### 2. تشغيل منصة نحكي
```bash
npm run dev
```

المنصة ستعمل على المنفذ 3000: `http://localhost:3000`

## 📱 استخدام نظام الإشعارات

### صفحات النظام:
- **الصفحة الرئيسية**: `http://localhost:3000` - جرس الإشعارات في الهيدر
- **صفحة الإشعارات**: `http://localhost:3000/notifications` - إدارة كاملة للإشعارات
- **جميع الصفحات**: جرس الإشعارات متاح في كل مكان

### ميزات النظام:

#### 🔔 **إشعارات لحظية**
- إشعارات فورية عبر Socket.IO
- اتصال مباشر بين المستخدمين
- تحديث فوري للعدادات والقوائم

#### 🌐 **إشعارات المتصفح**
- إشعارات نظام التشغيل
- تعمل حتى لو كانت المنصة في الخلفية
- أصوات وإشارات بصرية

#### 📋 **قائمة الإشعارات**
- عرض جميع الإشعارات
- فلترة حسب الحالة (مقروء/غير مقروء)
- إحصائيات مفصلة
- إجراءات مختلفة (تحديد كمقروء، حذف، مسح الكل)

## 🧪 اختبار النظام

### إرسال إشعار تجريبي:
1. اذهب إلى صفحة الإشعارات: `/notifications`
2. استخدم قسم "إشعار تجريبي" 
3. عدّل البيانات واضغط "إرسال إشعار تجريبي"
4. ستظهر الإشعارات فوراً في:
   - جرس الإشعارات (عداد أحمر)
   - قائمة الإشعارات
   - إشعار المتصفح (إذا كان مفعلاً)

### أنواع الإشعارات المدعومة:
- ❤️ **إعجاب**: عندما يعجب أحد بمنشورك
- 💬 **تعليق**: عندما يعلق أحد على منشورك  
- 👤 **متابعة**: عندما يتابعك مستخدم جديد
- 📢 **إشارة**: عندما يشير إليك أحد في منشور
- 📝 **منشور**: عندما ينشر أحد المتابعين منشوراً جديداً
- ⭐ **ترشيح**: عندما يرشحك مؤثر كـ "كبير متابعين"

## 🔧 إعدادات متقدمة

### متغيرات البيئة (.env):
```
NEXT_PUBLIC_SOCKET_URL=http://localhost:3001
```

### إعدادات الخادم (server/.env):
```
PORT=3001
ALLOWED_ORIGINS=http://localhost:3000,http://localhost:3001
```

## 🚨 استكشاف الأخطاء

### إذا لم تعمل الإشعارات:
1. تأكد من تشغيل خادم الإشعارات على المنفذ 3001
2. تحقق من حالة الاتصال في صفحة `/notifications`
3. تأكد من تفعيل إشعارات المتصفح
4. تحقق من console المتصفح للأخطاء

### إذا لم تظهر إشعارات المتصفح:
1. اضغط على "تفعيل إشعارات المتصفح" في صفحة الإشعارات
2. تأكد من السماح للإشعارات في إعدادات المتصفح
3. تحقق من أن المتصفح يدعم Notification API

## 📊 مراقبة النظام
- في console الخادم ستظهر رسائل الاتصال والإرسال
- في console المتصفح ستظهر رسائل حالة الاتصال
- عدادات الإشعارات تتحدث تلقائياً

نظام الإشعارات جاهز للعمل! 🎉
