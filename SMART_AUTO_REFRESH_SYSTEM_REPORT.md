# نظام التحديث التلقائي الذكي - منصة نحكي

## 📊 نظرة عامة

تم تطوير نظام تحديث تلقائي متقدم وذكي لضمان عرض البيانات الحديثة في جميع صفحات منصة "نحكي" بدون تدخل المستخدم.

## 🚀 الميزات الرئيسية

### 1. التحديث التلقائي الذكي
- ✅ **تحديث دوري**: كل 30 ثانية للمشرفين، كل دقيقة للوحة التحكم
- ✅ **توقف عند الإخفاء**: يتوقف التحديث عند إخفاء الصفحة لتوفير الموارد
- ✅ **تحديث عند العودة**: يحدث البيانات فور عودة المستخدم للصفحة
- ✅ **تحديث فوري**: إمكانية التحديث اليدوي في أي وقت

### 2. نظام Context للتحديث المتقاطع
```typescript
// يربط بين الصفحات المختلفة
const { triggerRefresh } = useDataRefresh();

// عند إنشاء مشرف جديد
triggerRefresh('admins'); // يحدث صفحة المشرفين
triggerRefresh('users');  // يحدث إحصائيات المستخدمين
```

### 3. Hook مخصص للتحديث (`useAutoRefresh`)
```typescript
const autoRefresh = useAutoRefresh(fetchData, {
  intervalMs: 30000,      // الفترة الزمنية
  enabled: true,          // تفعيل/إلغاء
  pauseOnHidden: true,    // توقف عند الإخفاء
  refreshOnFocus: true    // تحديث عند العودة
});
```

### 4. مؤشر حالة التحديث
- 🟢 **حالة النشاط**: مؤشر مرئي للتحديث التلقائي
- ⏰ **توقيت آخر تحديث**: عرض الوقت المنقضي
- 🔄 **العد التنازلي**: وقت التحديث القادم
- 📊 **إحصائيات**: عدد مرات التحديث
